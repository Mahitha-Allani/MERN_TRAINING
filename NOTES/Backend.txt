**BACKEND DEVELOPMENT**

HTTP Server--->APIs---->Database

HTTP Server: 
    -It is a server that accepts requests and sends responses.[HTTPRequests,HTTPResponse]
    -It cannot handle database operations.
    -HTTP server should be on state and have a port number.
    -HTTP Protocol is used to communicate between HTTP Server and APIs.

APIs: Application Programming Interfaces.
      -APIs have process to requests but cannot receive requests.
      -An Application can have multiple APIs.
      -APIs contains request handling mechanisms.
      -Protocol between API and database depends on the database used.
      -APIs are used to connect the database to the web application.
      -Handling,Authentication and Authorization are done in APIs.

Database: 
    -It is a collection of data stored in a structured way.
    -It is used to store and retrieve data.
    -Indexing is needed .
